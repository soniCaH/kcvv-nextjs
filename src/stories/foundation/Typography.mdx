# Typography

All type tokens are defined in `src/app/globals.css`. The body font is **Montserrat** (loaded via
Adobe Typekit); the heading font is **Quasimoda**. System-safe fallbacks are specified for all families.

---

## Font Families

| Token | Family | Usage |
|---|---|---|
| `font-title` | `quasimoda, acumin-pro, Montserrat, Verdana, sans-serif` | All headings (h1–h6) |
| `font-body` | `montserrat, -apple-system, system-ui, BlinkMacSystemFont, Segoe UI, Roboto, Helvetica Neue, Arial, sans-serif` | Body text (default) |
| `font-alt` | `stenciletta, …, sans-serif` | Accent / display |
| `font-mono` | `ibm-plex-mono, Consolas, Liberation Mono, Courier, monospace` | Code |

## Heading Scale

Headings are set globally in CSS — no per-component overrides needed.

| Heading | Mobile | Desktop (≥768px) |
|---|---|---|
| h1 | 28px (`4xl`) | 48px (`6xl`) |
| h2 | 22px (`2xl`) | 32px (`5xl`) |
| h3 | 20px (`xl`) | 24px (`3xl`) |
| h4 | 18px (`lg`) | 22px (`2xl`) |
| h5 | 16px (`base`) | 16px (`base`) |
| h6 | 13px (`sm`) | 13px (`sm`) |

---

## Font Weights

| Token | Value | Usage |
|---|---|---|
| `font-normal` | 400 | Body text |
| `font-medium` | 500 | Emphasis, labels |
| `font-bold` | 700 | Headings, CTAs |

---

## Line Heights

| Token | Value | Usage |
|---|---|---|
| `leading-tight` | 1.2 | Headings |
| `leading-snug` | 1.25 | Subheadings, blockquotes |
| `leading-normal` | 1.5 | Short body |
| `leading-relaxed` | 1.6 | Article content |
| `leading-loose` | 1.75 | Default body (`<body>`) |

## Special utilities

<code className="font-mono text-sm">const x = 1</code>
