## Breakpoints

| Token | Value | Notes |
|---|---|---|
| `sm` | 640px | Tailwind default |
| `md` | 768px | Foundation medium |
| `desk` | 960px | Gatsby parity breakpoint |
| `lg` | 1024px | Tailwind default |
| `xl` | 1280px | Foundation xlarge |
| `2xl` | 1536px | Tailwind default |

---

## Max Widths

| Token | Value | px | Usage |
|---|---|---|---|
| `max-w-inner` | 60rem | 960px | Article content, narrow layouts |
| `max-w-inner-lg` | 70rem | 1120px | Wide content areas |
| `max-w-outer` | 90rem | 1440px | Full-bleed container cap |

---

## Shadows

| Token | Usage |
|---|---|
| `shadow-sm` | Cards, subtle elevation |
| `shadow` | Default card shadow |
| `shadow-md` | Raised elements |
| `shadow-lg` | Modals, overlays |
| `shadow-green` | Focus glow (`0 0 6px 1px rgba(74,207,82,0.5)`) |
| `shadow-input` | Form input inset |
| `shadow-input-focus` | Form input focus state |

## Usage in code

```tsx
// Named import from @/lib/icons
import { Trophy, MapPin } from "@/lib/icons";
<Trophy className="w-5 h-5 text-kcvv-green" />

// Via Icon component (design-system wrapper)
import { Icon } from "@/components/design-system/Icon";
<Icon name="trophy" size="md" color="primary" />
```
